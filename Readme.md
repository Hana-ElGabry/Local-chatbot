# StarCoder2 3B Local Chatbot

## Step 1: Pull the Model
ollama pull starcoder2:3b

## Step 2: Install Requirements
pip install -r requirements.txt

## Step 3: Run the App
python app/main.py

Open http://localhost:7860 in your browser.

## Sample Prompts

1. Write a Python for loop that prints numbers 1 to 10
2. Create a Python function that adds two numbers
3. Write Python code to create a list and append 5 items to it
4. Write a Python if-else statement that checks if a number is positive
5. Create a Python function that reverses a string
